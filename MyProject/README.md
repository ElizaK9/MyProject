Проект: "Разработка сайта для запоминания корейских слов."

Этот проект представляет собой сайт с прохождением тестов для запоминания корейских слов по различным тематикам. 
На главной странице доступны для выбор тема, режим прохождения теста (с картинками и без) и количество слов в тесте. 

<img width="1872" height="938" alt="image" src="https://github.com/user-attachments/assets/6a5b8aec-8643-480f-b477-9cee2ddb6252" />

На вкладке "Коресйкий алфавит" приведены все буквы корейского алфавита, а также всплывающее окно при нажатии на букву с транскрипцией и правилами чтения.
